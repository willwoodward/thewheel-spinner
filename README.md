## How to Use

### Player Mode
Use this to randomly select which player goes next. Choose between:
- **2 Players** - Blue and White segments
- **3 Players** - Blue, Yellow, and White segments

### Quiz Mode
Use this for the quiz portion of the game with 7 segments:
- Gold, Blue, Red, Blue, Blue, Red, Blue

Simply press the **SPIN** button and watch the wheel go! The result will be displayed after the wheel comes to a stop.

## Running Locally

Simply open `index.html` in a web browser, or serve it with any static file server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve
```

Then open http://localhost:8080 in your browser.
